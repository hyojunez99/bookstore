# 3D Book Archive

## 프로젝트 한 줄 요약

CSS 3D Transform을 활용해 책을 3D 오브젝트처럼 회전시키는 인터랙션을 구현하고,  
직접 읽은 책의 한 줄 요약과 인상 깊었던 내용을 기록한 개인 아카이브형 웹 프로젝트입니다.

## 주요 기능

- CSS 3D Transform을 활용한 책 회전 인터랙션 구현  
- 마우스 및 버튼 조작에 따른 3D 시점 변화  
- 책 정보 아카이빙을 위한 콘텐츠 구조 설계  

---

## 기술 스택

- Frontend: HTML5, CSS3, JavaScript (ES6+)  

---

## 담당 역할

- 개인 프로젝트로 전체 UI/UX 설계, 3D 회전 인터랙션 구현 및 콘텐츠 구조 설계 담당

---

## 배운 점 / 인사이트

### 3D UI 구현 경험

3D UI 구현에서는 단순히 transform을 적용하는 것보다 DOM 구조와 레이어 분리를 먼저 설계하는 것이 중요하다는 점을 배웠습니다.

### 유지보수와 확장 고려

시각적 효과를 구현할 때도 유지보수와 확장을 고려한 구조 설계가 필요하다는 기준을 갖게 되었습니다.
